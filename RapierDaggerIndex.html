How to read this table

  • Ranks AU/NZ longsword fencers for local seeding and near-term match prediction, rather than published global rating alone.
  • Offers four views of the same Glicko-style data: raw rating, conservative lower-bound seeding, balanced local-centre shrinkage, and an optimistic upside model.
  • Interprets the source dataset’s confidence value as Glicko-2 rating deviation (RD), where lower values mean higher trust and higher values mean more uncertainty.
  • Raw uses published rating only, with no uncertainty adjustment.
  • Conservative discounts uncertain ratings to produce a more cautious ordering.
  • Balanced pulls uncertain ratings toward the local median, tempering extreme uncertain placements.
  • Optimistic gives uncertain improvers a bounded upside bonus while staying anchored to the published rating.
  • Change shows how far the displayed skill moves from the published rating.
  • Vs Median estimates the chance of beating a synthetic reference fencer built from the selected comparison pool, using both displayed skill and both sides’ uncertainty.
  • The alternative modes are analytical lenses on the same rating base, not replacements for the underlying Glicko-2 style rating.
  • Groups athletes into quantile-based letter tiers, with + / – markers indicating whether a fencer sits near the top or bottom of their tier.
  • Uses the ordering S, A, B, C, D, F for the first six tiers; if more tiers are added later, the next tier is 7, followed by Tier 8 and so on.
  • Uses a heatmap to make relative strength across the field easier to compare at a glance.
  • Shows uncertainty (σ) using the underlying Glicko-style deviation, where lower values indicate a more reliable estimate.
Optimistic mode gives a bounded uncertainty-based upside bonus while staying anchored to published ratings. Vs Median estimates performance against the selected reference pool using both displayed skill and both sides’ uncertainty.
