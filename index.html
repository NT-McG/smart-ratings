AU-NZ Longsword – Improved Local Rating

How to read this table

  • Ranks AU/NZ longsword fencers for local seeding and near-term match prediction, rather than published global rating alone.
  • Supports both the raw HEMA Rating and an improved local rating that gives controlled upside to higher-uncertainty ratings.
  • Interprets the source dataset’s confidence value as Glicko-2 rating deviation (RD), where lower values mean higher trust and higher values mean more uncertainty.
  • Uses a diminishing-returns bonus based on RD, with tunable strength and scale, so uncertainty can raise a fencer’s local seeding estimate without letting sparse data dominate the table.
  • Bonus strength controls how much the improved local rating rewards uncertainty overall. Higher values make high-RD competitors climb faster; lower values keep the table closer to the raw HEMA Rating.
  • Bonus scale controls how quickly that uncertainty bonus ramps up. Lower values make the bonus kick in earlier and more sharply; higher values make it gentler and require more RD before a large uplift appears.
  • A practical way to tune them is to start with moderate values, compare the leaderboard to recent local results, then raise bonus strength or lower bonus scale if improving newcomers still look underrated, or do the reverse if sparse-data fencers are being pushed too high.
  • Optionally applies an underdog factor so lower-rated improvers get a little more upside than already-established top players.
  • Groups athletes into quantile-based letter tiers, with + / – markers indicating whether a fencer sits near the top or bottom of their tier.
  • Uses the ordering S, A, B, C, D, F for the first six tiers; if more tiers are added later, the next tier is F-, followed by Tier 7+, Tier 7, Tier 7-, Tier 8+, and so on.
  • Uses a heatmap to make relative strength across the field easier to compare at a glance.
  • Shows uncertainty (σ) using the underlying Glicko-style deviation, where lower values indicate a more reliable estimate.
  • Includes Win % as the expected result against the dataset’s average-strength fencer, providing a simple baseline for competitive advantage.
Raw HEMA Rating uses published rating only. Improved local rating adds a capped RD-based bonus; enabling underdog boost gives slightly more uplift to lower-rated improvers.
